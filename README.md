# Plataforma ITAVU

Plataforma basada en usuarios - aplicaciones, en la cual se programan las necesidades del instituto para solventar la administracion de cada departamento.

![alt text](https://github.com/printepolis/itavu/blob/master/readme.png)

## Principales caracteristicas, en él area de desarrollo:

- Interfaz grafica integrada, responsive y unificada
- Capa de seguridad para entorno https. 
- CSS separado para Android, Iphone y IPad
- Gestion via correo. Envia una notificacion y correo con una sola linea
